Курси розробки ігор на Java - 2019

18-21 червня 2019 року учні 7-х класів продовжили навчання на курсі з розробки ігор мовою Java на базі Вінницького національного технічного університету. Грищук Тетяна Вікторівна, викладач факультету комп'ютерних систем і автоматики, поглибила знання та навички з ООП.
